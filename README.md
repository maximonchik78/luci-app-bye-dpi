# LuCI ByeDPI Manager

Универсальное Luci приложение для управления ByeDPI-OpenWrt.

## Возможности

- Автоматическая установка ByeDPI с определением архитектуры
- Автоподбор стратегий (как в ByeDPIManager)
- Управление кастомными стратегиями
- Тестирование доступности сайтов
- Полная интеграция с Luci
- Единый размер всех кнопок
- Поддержка OpenWrt 24.x и новее

## Установка

### 1. Сборка из исходников

```bash
git clone https://github.com/yourusername/luci-app-bye-dpi.git
cd luci-app-bye-dpi
# Добавьте пакет в дерево OpenWrt
cp -r . /path/to/openwrt/package/network/services/luci-app-bye-dpi
# Соберите пакет
make package/luci-app-bye-dpi/compile
2. Установка готового пакета
Скачайте IPK файл для вашей архитектуры

Загрузите на роутер

Установите:

bash
opkg install luci-app-bye-dpi_*.ipk
Использование
Откройте Luci веб-интерфейс

Перейдите в раздел Сервисы → ByeDPI Manager

Используйте кнопки для управления:

Установить/Обновить - автоматическая установка ByeDPI

Автоподбор стратегий - тестирование стратегий на выбранных сайтах

Применить стратегию - использование кастомной стратегии

Требования
OpenWrt 24.x или новее

Доступ в интернет для скачивания пакетов

Минимум 16MB свободного места

Поддерживаемые архитектуры
x86_64

aarch64_generic

arm_cortex-a15_neon-vfpv4

mipsel_24kc

Сборка на GitHub
Пакет автоматически собирается при пуше в ветку main и создании тегов.

Ручной запуск сборки:
Перейдите в Actions

Выберите "Build LuCI ByeDPI App"

Нажмите "Run workflow"

Выберите версию OpenWrt и архитектуры
